# 📒 随心记 - 智能笔记应用

一个功能强大的个人笔记管理系统，支持AI智能功能和社区分享。

## ✨ 主要功能

### 📝 笔记管理
- **创建笔记**：支持Markdown格式，实时预览
- **编辑笔记**：所见即所得的编辑器
- **删除笔记**：支持软删除，可从回收站恢复
- **笔记历史**：自动保存版本历史，支持回滚

### 🤖 AI智能功能
- **AI摘要**：一键生成笔记核心要点
- **AI问答**：基于笔记内容进行智能对话
- **AI标签推荐**：智能推荐合适的标签

### 🔍 搜索与组织
- **全文搜索**：支持标题和内容搜索
- **标签系统**：为笔记添加标签，便于分类
- **文件夹管理**：创建文件夹组织笔记
- **高级筛选**：按文件夹、标签组合筛选

### 🌐 笔记广场
- **公开分享**：将笔记设为公开分享
- **社区浏览**：浏览其他用户的公开笔记
- **评论互动**：在公开笔记上进行评论
- **广场搜索**：在公开笔记中搜索内容

### 📁 数据管理
- **Markdown导入**：支持从.md文件导入笔记
- **Markdown导出**：将笔记导出为.md文件
- **数据安全**：用户数据隔离，保护隐私

### 👤 用户系统
- **用户注册**：安全的用户注册系统
- **个人资料**：修改用户名和密码
- **权限控制**：严格的访问权限管理

## 🚀 快速开始

1. **注册账号**：访问应用首页注册新用户
2. **创建笔记**：点击"写笔记"开始记录
3. **使用AI功能**：在笔记详情页使用AI摘要和问答
4. **分享笔记**：将优质笔记设为公开，分享到广场
5. **探索广场**：浏览其他用户的精彩笔记

## 🎯 使用技巧

- 使用标签系统更好地组织笔记
- 定期查看笔记历史，避免丢失重要修改
- 在广场分享笔记时，可以获得其他用户的反馈
- 利用AI功能快速理解和整理笔记内容

## 📋 技术栈

- **后端**：Flask + SQLAlchemy
- **前端**：Bootstrap 5 + JavaScript
- **AI服务**：集成DeepSeek AI
- **数据库**：SQLite
- **部署**：Python虚拟环境
